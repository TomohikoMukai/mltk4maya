# MLTK4Maya
Autodesk Maya向けの簡易的な機械学習ツールです．今のところ，線形回帰と特異値分解を用いた次元削減のみ提供しています．

## インストール方法
* plug-insフォルダにあるファイル一式を、Mayaのプラグインパス（MAYA_PLUG_IN_PATH）が通っているフォルダに置きます。
* scriptsフォルダにあるファイル一式を，Mayaのスクリプトパス（MAYA_SCRIPT_PATH）が通っているフォルダに置きます。
